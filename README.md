
## Installation
Use python 3.6>= x <9
```
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```
You may need to install tensorflow separately as well.




## Running experiments
```
python run.py --help
```

## Generating plots
```
python analyze.py --help
```
